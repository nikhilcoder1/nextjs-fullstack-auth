# Next.js Full Stack Authentication System

A production-grade authentication system built using Next.js and MongoDB.

## Features
- User registration & login
- Password hashing
- JWT-based authentication
- Protected routes
- Email verification
- Forgot & reset password
- Email testing using Mailtrap

## Tech Stack
- Next.js
- MongoDB
- Mailtrap
- JWT
- bcrypt

## Getting Started

```bash
npm install
npm run dev
